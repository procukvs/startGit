      Проект - «Факультет»
Задача –інформаційна підтримка учбового процесу і організаційної
діяльності на факультеті вузу. 
Програма повинна містити учбовий план, розклад занять, списки 
груп студентів, списки викладачів, що працюють по кафедрам, 
і списки аспірантів(з керівниками і темами досліджень). 
Програма повинна забезпечити складання:
•	Розкладу занять на семестр (по групам);
•	Учбового плану (по семестрам) для кожного курсу;
•	Розклад занять для викладачів;
•	Список телефонів співробітників;
•	Навантаження по годинам для викладачів;
•	Списки наукових кадрів (викладачів, аспірантів) за науковими напрямками;
•	Списки студентів-дипломників (по групам і по викладачам).
